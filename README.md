Для начала нужно клонировать репозиторий и установить библиотеки для работы, потому введем pip install aiogram aiohttp pandas matplotlib.
Чтобы можно было сделать нового бота, нам нужен @BotFather в Telegram. Заходим, копируем оттуда токен и вставляем его в TELEGRAM_TOKEN вместо имеющегося.
Теперь мы сделали для работы всё, что нужно.
Функционал:
  /start - Запускает бота и выводит приветственное сообщение.
  /help - Отображает список доступных команд и их описание.
  /song - Позволяет выполнить поиск песен по названию.
    (Бот обращается к Deezer API, выводит до 20 результатов, показывает их по 3 за раз, поддерживает кнопку «Показать ещё».)  
  /artist - Позволяет получить информацию об исполнителе
    (Имя, количество фанатов, ссылка на страницу Deezer, фотография исполнителя.)
  /info - Выводит расширенную информацию о треке
    (Название, исполнитель, альбом, длительность, рейтинг Deezer, ссылка на трек.)
